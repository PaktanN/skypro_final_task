Git

  • git init - инициализация нового репозитория.
  • git clone <адрес> - клонирование существующего репозитория.
  • git add <файл/папка> - добавление изменений в индекс.
  • git commit -m "сообщение" - фиксация изменений с сообщением.
  • git push - отправка изменений на удаленный сервер.
  • git pull - получение и интеграция изменений с удаленного сервера.
